# Proyecto_Final
Funcionalidades Mínimas
▶ El acceso de diversos perfiles (admin, registrado, etc).
▶ Cargar un nuevo post, eliminar post publicados (con diversas
restricciones).
▶ Comentar posts por parte de usuarios con perfil registrado. Un usuario
debe estar autenticado y tener perfil registrado para poder comentar.
▶ Login a usuario tipo admin y registrado.
▶ Filtrar post por fecha, categoría de post y comentarios recibidos.


REQUERIMIENTOS MINIMOS

URL DEL REPO
URL DEL DEPLOY
URL DEL VIDEO
